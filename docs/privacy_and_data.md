# Privacy And Data

This public repository is designed for demonstration and reproducibility of the workflow, not for distributing the full private manuscript or source corpus.

## Included

- Core pipeline scripts.
- Prompt templates.
- A public demo configuration.
- Synthetic examples.
- Documentation and visuals.

## Excluded

- Complete generated manuscript files.
- Full PDF and full LaTeX output.
- Raw paper PDFs.
- Full paper metadata pools.
- API keys, tokens, and `.env` files.
- Local machine paths.
- Personal emails and affiliation metadata.
- Obsolete snapshots and compile artifacts.

## Reproducing A Full Run

To run the complete pipeline, provide your own paper metadata pool and bibliography, then update `config/pipeline_config.json`. Keep private datasets outside Git, or publish only a documented, license-compatible sample.
