# Examples

The `examples/` directory contains synthetic, shortened files that show the expected shape of inputs and outputs.

| File | Role |
| --- | --- |
| `sample_papers.csv` | Tiny paper metadata pool. |
| `sample_references.bib` | Minimal BibTeX entries. |
| `sample_chapter.md` | Example generated chapter excerpt. |
| `sample_section.tex` | Example LaTeX section output. |
| `sample_output_preview.md` | High-level preview of pipeline outputs. |

These are intentionally small. They are for orientation, not for reproducing the full private survey.
